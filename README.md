# Registry Hospital Database

Цей репозиторій містить навчальний проєкт з проєктування та реалізації
реляційної бази даних у межах курсу **“Вступ до баз даних”**.

Предметна область проєкту — зберігання та обробка прийомів
та додавання нових спеціалізацій і лікарів до реєстратури лікарні.
